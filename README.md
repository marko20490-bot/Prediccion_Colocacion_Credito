# Prediccion_Colocacion_Credito
Análisis predictivo de la colocación de créditos de bancos privados grandes en Ecuador entre 2005 y 2025, mediante modelos ARIMAX, para identificar patrones históricos, evaluar el impacto de variables macroeconómicas y generar proyecciones que apoyen una gestión eficiente y sostenible del crédito.
